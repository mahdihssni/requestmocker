# Privacy Policy (Request Mocker)

Request Mocker does **not** collect, transmit, or sell personal data.

## What data is stored

- Route definitions you create (match rules, mock body, headers, status code, delay)
- Local monitoring stats (hit counts and recent mocked request events)

This data is stored **locally on your device** using Chrome extension storage (`chrome.storage.local`).

## What data is shared

- No data is sent to any server by this extension.

## Permissions justification

- **storage**: Save your routes and monitoring stats locally.
- **host_permissions `<all_urls>`**: Allows the extension to apply your mock rules on any site you choose to use it on.

If you have questions, contact the publisher listed on the Chrome Web Store page.
